# ChainLogistics SDK Documentation

This directory contains shared SDK documentation for the Python and Rust SDKs.

## Documentation Index

- `INSTALLATION.md`: installation and setup for Python and Rust SDKs.
- `API_REFERENCE.md`: high-level API reference by service and method.
- `EXAMPLES.md`: common end-to-end usage examples.
- `MIGRATION_GUIDE.md`: migration guidance for SDK upgrades.
- `python/README.md`: Python SDK details.
- `rust/README.md`: Rust SDK details.

## Supported SDKs

- Python: `sdk/python`
- Rust: `sdk/rust`
